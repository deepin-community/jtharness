You can run the JT4Demo two different ways:
  • On the same computer that runs the JavaTest harness (in a separate JVM) by means of the Runtime.exec Java API method.
  • On another computer or device using a JavaTest Agent. The JavaTest Agent runs tests on a separate computer and communicates with the JavaTest harness over the network. Choose this answer if you cannot run your tests on the same computer that runs the JavaTest harness.
